# Currency Converter App
## Description

This is the first challenge from our new and upcoming. This app will be used to convert 200+ supported currencies from one to another. The "Currency Code" field (ex. USD) follows the standard _ISO 4217 Three Letter Currency. This ensures clarity and reduces errors in any exchange activities.

The app is a _Next.JS_ <sup>[2]</sup> project which utilizes the  feature and uses the third-party data fetching service, _SWR_  (_Fun Fact! The developers of NextJS also made this service_). This makes sure that data will always be available whenever you use this app.

---

## Try my project!

## Limitations

Currently, these are the limitations of this app:

1. The data updates only once a day.
2. The currency name is not returned, only the code (i.e. USD, PHP). It would be better if you can see the currency name alongside the code for a much better user experience.

These limitations can be overcome either by upgrading the plan or by finding another API service which returns the currency name.
